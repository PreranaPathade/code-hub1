| DevOps Stage | Jira Use                                                          |
| ------------ | ----------------------------------------------------------------- |
| **Plan**     | Create user stories, sprints, and tasks.                          |
| **Code**     | Link Jira issues with Git commits or pull requests.               |
| **Build**    | Automatically update Jira when builds succeed/fail (via Jenkins). |
| **Test**     | Log bugs from testing tools like Selenium or Zephyr.              |
| **Deploy**   | Track deployment tickets or automation pipelines.                 |
| **Monitor**  | Create incidents/issues when problems are found in production.    |
